This project implements a deep learning-based image classification model that identifies different types of flowers from images. The model uses a Convolutional Neural Network (CNN) built with TensorFlow/Keras to automatically learn visual features and classify flower images into multiple categories.

👾 Features

Automatic flower image classification

Uses Convolutional Neural Networks (CNN) for feature extraction

Image preprocessing and normalization

Dataset split into training and validation sets

Visualizes sample images and predictions

👀 Objectives

✓ Build an image classification model using deep learning
✓ Automatically identify different flower species from images
✓ Apply CNN techniques for feature extraction and classification
✓ Develop a practical computer vision application

🚀 Model Details

CNN (Convolutional Neural Network):
Extracts spatial features from flower images using convolutional layers.

Image Preprocessing:
Images are resized and normalized before training the model.

Dataset Split:
Training and validation sets are used to evaluate model performance.

Approach Type:
Deep Learning – Image Classification
